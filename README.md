# 🧇 Waffle Bar Restaurant Website

A modern, clean, and fully responsive restaurant website for Waffle Bar, built from scratch using HTML5 and CSS3.

## 🚀 Live Demo
https://st18779.github.io/waffle-bar-restaurant/

---

## ✨ Features
- 📱 **Fully Responsive:** Optimized for all screen sizes (Desktops, Tablets, and Mobile phones).
- 🎨 **Modern Design:** Clean layout with a focus on high-quality food imagery.
- 📜 **Interactive Menu:** Clean and organized sections displaying the restaurant's dishes.
- 📍 **Contact & Location:** Clear details for reservations and finding the restaurant.

---

## 🛠️ Technologies Used
- **HTML5:** For structured and semantic web content.
- **CSS3:** For modern styling, layouts (Flexbox/Grid), and responsive media queries.
